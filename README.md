# My-Tugas
[Academic Peoject: OOP Class] Assignment Tracking App for Android Built with JetpackCompose


## Features
- Create and manage tasks with details like titles, descriptions, and due dates.
- Edit or delete existing tasks as needed.
- Automatic updates for task status, including marking overdue assignments.
- Notification system to remind users of upcoming deadlines.
- Calendar view to see tasks by date, with visual indicators for quick reference.
- User profile options, such as changing your username or profile picture.
- Statistics screen to review completion rates and overall progress.
- Smooth navigation and subtle animations for a better user experience.

## How to Install
1. Go to the release
2. Download the .Apk
3. Install the app 
4. done

Note: Make sure to allow permission on your phone for installing outside Playstore. Also Google Play Protect gonna request for scanning the app since its not from playstore, just do it
